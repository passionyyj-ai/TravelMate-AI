# TravelMate AI v2.0

## 고정 주소
앞으로 아래 주소만 사용합니다.

https://passionyyj-ai.github.io/TravelMate-AI/

버전 번호를 URL에 붙이지 않습니다. QR 코드와 홈 화면 아이콘도 계속 동일하게 사용합니다.

## 반영 기능
- 앱 안에서 QR 코드 표시
- 휴대폰 공유 메뉴 및 주소 복사
- Android PWA 설치 버튼
- iPhone 설치 단계 안내
- 새 서비스 워커가 감지되면 `지금 업데이트` 표시
- 업데이트 후 여행 데이터와 로그인 정보 유지
- 오프라인 실행에 필요한 핵심 파일 캐시

## GitHub Pages 배포
ZIP의 모든 파일을 GitHub 저장소 루트에 덮어쓴 뒤 커밋합니다.
기존 아이콘 파일명과 고정 URL은 변경하지 마세요.

## 업데이트 테스트
1. 기존 앱을 실행합니다.
2. GitHub에 새 버전을 올립니다.
3. 앱을 다시 열거나 백업 화면을 엽니다.
4. 새 버전 안내가 나타나면 `지금 업데이트`를 누릅니다.
